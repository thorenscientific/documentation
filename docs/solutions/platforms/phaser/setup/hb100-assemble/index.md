# HB100 X Band Signal Source Assembly

 An HB100 signal source is included with the Phaser kit.  For many of the beamforming labs, we use it as a simple x band (roughly 10.2 to 10.4 GHz) signal source.  More information here:  
 
 [DFRobot](https://www.dfrobot.com/product-1403.html?srsltid=AfmBOoqxvUziIfRxUZjQMj95TpL07p1cqWjNkIDSNVv9eDUqzaD3e4KP)

 ```{image} hb100.svg
:alt: Phaser Picture
:width: 200px
```
 
 It is powered with a 3-5V supply -- most commonly we use 3 AA batteries in series.  

 There are many ways to mount this, such as a cell phone grip holder.  But if you would like to attach it to a small tripod, then [this step file](../../hardware/schematics/HB100_holder.stl) can be sent to a 3D printer.  

It is a case for the HB100 that allows you to hold it with a standard a 1/4-20 thread camera mount. 

For those wishing to take this a step further, you can also mount a battery pack to it.  The parts used for this can be found at these links:
* [Battery pack](https://a.co/d/56lOPHX)
* [Connectors to modify the battery pack wires](https://a.co/d/dlMQT0t)
