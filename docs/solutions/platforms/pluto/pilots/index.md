A **Platform Pilot** is more than just a demo!  It is a fully documented system-level example of a representative use case--complete with all the key elements to implement that use case.  Each PlutoSDR Pilot will contain:

-   Background training and theory on what is being accomplished
-   Hardware + software setup
-   Example scripts in Python, MATLAB, or QIQ Systems Software
-   Documented workflows (setup → configure → run → analyze → modify)
-   A foundation for your algorithm development
